# P.5

- IP란?: Internet Protocol. IP 주소란 인터넷 상의 컴퓨터가 가지고 있는 고유 주소이다.
    - 도메인(Domain)이란? 네 개의 8비트 숫자로 표현되어 있는 IP 주소를 문자 형태로 변환한 형태. 문자 형태의 도메인을 실제 IP 주소로 바꿔 주는 서버가 DNS 서버이다.
